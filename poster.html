04 / Advanced Engineering Features

What Makes This Different

Zero-RAM Stateful UI
Navigation breadcrumbs persisted in Supabase ui_nav_stack JSONB column. Survives server restarts and cold-starts completely.
_db_push_nav_stack / _db_pop_nav_stack
Dual-Node AI Redundancy
Gemini 2.5 Flash as primary reasoning engine. Automatic 429/quota failover to Groq Llama-3-70B with zero user disruption.
ResourceExhausted → groq_chat_fallback()
Smart Contextual Voice Routing
AI injects <VOICE_REQUIRED> tags dynamically. System routes to TTS (Edge-TTS synthesis) or text based on content type & user preference.
voice_pref: text | voice | smart_routing
Dynamic Timezone Resolution
Accepts Telegram Location Pin (via TimezoneFinder) or routes typed text like "Karachi" through Groq LLM → strict IANA string → persisted to DB.
parse_timezone_with_groq() → Asia/Karachi
Natural Language Scheduling
dateparser resolves inputs like "tomorrow morning" or "next Monday at 3pm" into UTC. APScheduler fires automated dispatch crons.
AWAIT_SCHEDULE_TIME → dateparser.parse()
Human-In-The-Loop (HITL)
Before any email send, AI autonomously detects missing fields (e.g., recipient email) and prompts user for confirmation via inline keyboard — no hallucinated sends.
[Specify Recipient Email] sentinel pattern
